# 問題

## 1.

正の整数Nが与えられます. 
以下の条件を満たすような整数(a,b,c)の組の個数を求めて下さい.

1. sqrt(a)+sqrt(b)=sqrt(c)
2. 1<=a,b,c<=N

### 制約

1<=N<=10^5

### 入力
入力は以下の形式で与えられます.

```
N
```

### 出力

条件を満たす(a,b,c)の組の個数を出力してください.

### 入力例1

```
9
```

### 出力例1

```
4
```

(a,b,c)=(1,1,4),(1,4,9),(4,1,9),(2,2,8)の4つがあります.

解答コードは1.cpp
